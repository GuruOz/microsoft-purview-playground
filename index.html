Purview PowerShell format
  • Import: paste JSON from Get-DlpComplianceRule to visualise an existing rule.
  • Export: generates JSON ready for New-DlpComplianceRule — use with caution in production.
Visualizer workspace format
  • Import: restore a previously exported Purview Playground workspace.
  • Export: save your current workspace for backup or sharing — not for Purview PowerShell.

Production safety warning — read before using this JSON

  • Applying this in Purview PowerShell can modify or break live DLP enforcement in your production tenant.
  • Back up your current configuration first (e.g. export existing rules and policies with Get-DlpComplianceRule / Get-DlpCompliancePolicy).
  • Review every rule and test in a non-production environment where possible.
  • Use entirely at your own risk — you are solely responsible for any changes made to your environment.
